# Iris Dataset Analysis - Task 1

This project is part of my internship. It demonstrates basic Data Exploration and Visualization using Python.

## Objective
- Load and inspect the Iris dataset.
- Perform statistical summarization.
- Visualize data using Scatter plots, Histograms, and Box plots.

## Tools Used
- Python
- Pandas
- Matplotlib & Seaborn

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install pandas seaborn matplotlib`
3. Open `iris.ipynb` in Jupyter Notebook or VS Code.
